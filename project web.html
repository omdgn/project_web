About CarPI

CarPI (Creative Car Builder and Editor AI) is a next-generation, web-based application designed to empower users in creating, customizing, and exporting car designs using artificial intelligence and manual editing tools.

The system is developed with a modular architecture: the frontend is built in ReactJS, the backend uses Node.js and Python (FastAPI), and PostgreSQL is utilized for data management. For AI-driven operations, CarPI integrates powerful machine learning models such as GANs and CNNs alongside DALL·E API and PyTorch.

Users can select predefined car models and customize them in two ways: using AI-generated designs via descriptive prompts, or manual editing tools to modify features like color, rims, and accessories. All user-generated designs can be saved, shared, or exported in PNG, JPEG, or PDF formats.

CarPI also supports multilingual interfaces, real-time design previews, and role-based access for features like design history, premium exports, and user-specific customization workflows. Its scalable architecture ensures a seamless experience for both casual users and automotive design enthusiasts worldwide.
